## 📌 Overview

The **Police Case Reporting Portal** digitizes the complaint registration and tracking process for citizens. It eliminates paperwork, enhances transparency, and improves communication between the public and police authorities.

Built with pure **HTML5, CSS3, and Vanilla JavaScript**, this frontend prototype follows a modern **Glassmorphism** design language and is fully responsive across all devices. It's ideal for academic projects, UI/UX showcases, and can be easily integrated with a backend (Node.js, PHP, Django, etc.) for full-stack deployment.

## ✨ Features

### 🔐 Citizen-Facing Features
- **Online Complaint Registration** – Step-by-step form with incident details, personal information, and evidence upload.
- **Real-time Case Tracking** – Visual timeline showing each stage of the complaint (Submitted → Review → Investigation → Resolved).
- **User Dashboard** – Overview of complaint statistics, recent cases, and quick action cards.
- **My Complaints** – Table view of all filed cases with search and filter options.
- **Profile Management** – View and update personal details, change password.
- **Notifications** – Stay updated on case progress and police announcements.
- **Emergency Contacts** – Quick dial numbers (112, 100, 108) directly on the home page.
- **FAQ & Contact** – Searchable accordion FAQs and a contact form for inquiries.

### 🧑‍💼 Admin Features (UI Ready)
- Admin dashboard with complaint statistics (total, pending, under investigation, resolved, rejected).
- Recent complaints table with status badges and view actions.
- Search, filter, and assign officer UI placeholders.

### 🎨 Design & UX
- **Glassmorphism UI** with blurred, semi-transparent cards and backgrounds.
- **Smooth Animations:** Page loader, scroll reveal, floating background shapes, ripple effect, counter animation, and hover transitions.
- **Responsive Design:** Optimized for desktop, tablet, and mobile.
- **Accessibility:** Semantic HTML5, ARIA labels, keyboard navigation, high-contrast support.
- **Modern Color Palette:** Blue gradient theme with accent colors for a trustworthy government look.

### 🔒 Security UI Features
- Password strength indicator during registration.
- Login validation with toast notifications.
- Mandatory declaration checkbox before complaint submission.
- Placeholder for OTP verification, captcha, and session timeout screens.
